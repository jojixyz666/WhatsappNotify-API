Endpoints

GET/api/whatsapp/status

Returns current connection state. Possible values: CONNECTED, CONNECTING, DISCONNECTED, SCAN_QR.

{ "status": "CONNECTED" }
POST/api/whatsapp/send

Send a text notification via JSON body.

{ "to": "628123456789", "message": "Hello!" }
GET/api/whatsapp/send

Send via URL query string — useful for quick integrations and webhooks without a request body.

?to=628123456789&message=Hello+World
POST/api/whatsapp/webhook

Generic webhook receiver. Auto-decodes alerts from Grafana, Uptime Kuma, Netdata, etc. Falls back to DEFAULT_RECIPIENT env if to is omitted.

Supported body keys: to, phone, message, msg, text, content, title
GET/api/whatsapp/qr

Returns base64 QR image when session is in SCAN_QR state.

{ "qr": "data:image/png;base64,..." }
POST/api/whatsapp/logout

Clears session credentials and restarts the connection flow.

{ "success": true }
